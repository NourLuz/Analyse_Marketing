# 📊 Analyse Marketing – Projet Data Analyse & Dashboard

Projet complet d'analyse de données marketing, de l'exploration des données jusqu'à la création d'un tableau de bord interactif sous Power BI.

---

## 📁 Structure du projet

| Dossier | Contenu |
|---------|----------|
| `Données/` | Dataset source de l'analyse |
| `Python/` | Notebook Jupyter contenant tout le code |
| `Tableau de bord/` | Fichier Power BI (.pbix) du dashboard |
| `Captures du tableau de bord/` | Captures d'écran du rendu final |

---

## 🎯 Objectif

Analyser les performances des campagnes marketing, identifier les canaux les plus performants, segmenter les clients et fournir des recommandations actionnables à travers un dashboard interactif.

---

## 🔍 Étapes de l'analyse

1. **Nettoyage et préparation des données** – Traitement des valeurs manquantes, correction des types, création de variables
2. **Analyse exploratoire (EDA)** – Visualisation des tendances, distribution des ventes, saisonnalité
3. **Analyse par canaux marketing** – Performance comparée des différents canaux d'acquisition
4. **Segmentation clients** – Regroupement par comportement d'achat
5. **Recommandations** – Synthèse des insights pour la stratégie marketing
